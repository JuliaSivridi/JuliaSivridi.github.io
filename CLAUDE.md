# Julia Sivridi — Portfolio Website

## Что это
Одностраничный сайт-портфолио на GitHub Pages.

## Стек
- Платформа: Web (GitHub Pages)
- Язык: HTML, CSS, JavaScript
- Фреймворк: нет
- БД: нет

## UI система
Material Design 3. Иконки: Material Icons Outlined.
Цветовая схема: нейтральная/профессиональная, на усмотрение.

## Архитектура
Один HTML файл (или минимум файлов). Никаких фреймворков, никакого бэкенда.
Переключение языков EN/FI через JS без перезагрузки страницы.

## Принятые решения
- Без фото
- Без секции опыта работы
- Только секции: Hero, Projects, Contact
- Финский перевод генерирует Claude Code

## Что нельзя менять без обсуждения
- Структура секций
- Список проектов и ссылки

## Рабочий процесс
- Коммиты: атомарные, на английском
- После каждого этапа: отметить задачи в docs/TASKS.md, добавить запись в docs/WORK_LOG.md
- Обновлять этот файл при появлении новых архитектурных решений
