# juliasivridi.github.io

Personal portfolio site — [juliasivridi.github.io](https://juliasivridi.github.io)

## Stack

Plain HTML + CSS + JS. No frameworks, no backend. Hosted on GitHub Pages.

## Features

- EN / FI language switch (no page reload)
- Light / dark theme (follows system preference)
- Responsive layout

## Structure

```
index.html   — markup
style.css    — styles (Material Design 3, CSS custom properties)
script.js    — i18n
favicon.png
```
